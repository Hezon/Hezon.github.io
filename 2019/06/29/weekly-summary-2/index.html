JavaScript数据结构与算法学习总结一

发表于 2019-06-29 | 分类于 技术周报

数据结构

栈

stack 具有两项基本操作：

  • push，添加元素到栈的末尾（顶端）
  • pop，移除栈末尾（顶端）的元素

以上的两个操作可以概况为：后进先出，LIFO（last in, first out）。

另外，还有一些其他操作，如：

  • peek，获取末尾元素
  • isFull, 是否满栈
  • isEmpty，是否为空栈
  • size，栈的尺寸
  • clear，清空栈

使用数组实现一个栈类

队列

queue 具有两项基本操作：

  • enqueue，添加元素到队列的后端
  • dequeue，移除队列前端的元素

以上的两个操作可以概况为：先进先出，FIFO（first in, first out）。

另外，还有一些其他操作，如：

  • front，获取前端的元素
  • isEmpty，是否为空队列
  • size，队列的长度
  • clear，清空队列

使用数组实现一个队列类

链表

链表是数据元素的线性集合，每个元素由一个存储元素本身的节点和一个指向下一个元素的引用组成。

相对数组，链表添加或移除元素的时候不需要移动其他元素，能更有效地从序列中的任何位置插入或删除元素。

数组的好处是可以直接访问任何位置的任何元素，而要想访问链表中间的一个元素，需要从起点开始迭代列表直到找到所需的元素。

集合

集合有两个特性：

  • 集合中的成员是无序的
  • 集合中的成员都是不同的
